Introduction

東京都在住のフロントエンドエンジニアです。
コーポレートサイト制作などのWEB制作業務を幅広く対応いたします。
静的ページだけでなく、WordPressも対応可能です。
CSSアニメーションやJavaScriptを使った動きのあるサイト制作が得意です。

  • Skills

    HTML5, CSS3, Sass, JavaScript, jQuery, React.js, Redux, GSAP, PHP, WordPress, Git,Gulp,Webpack

  • Career

    体育大卒業後、カナダに長期滞在し就労経験を積みながら語学力を磨く。
    帰国後は専門商社にて営業職を経験する。語学力を活かし、海外工場との輸出入折衝や海外クライアントとの商談等をメインで担当する。
    その後、Web制作に興味を持ちWeb業界へ転職する。
    転職後はLP制作/コーポレートサイト制作を中心に、保守運用、協力会社の案件進捗管理など幅広い業務を担当する。
